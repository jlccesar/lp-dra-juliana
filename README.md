# Dra. Juliana Jardim — Site

Landing page e política de privacidade da Dra. Juliana Jardim — Estética e Harmonização Facial Avançadas (Brasília · Maceió · Recife).

Site estático (HTML/CSS/JS, sem build). Hospedado na Vercel.

## Estrutura

- `index.html` — homepage (hero, tratamentos, sobre, depoimentos, FAQ, localização, CTA)
- `politica-de-privacidade.html` — política de privacidade
- `image-slot.js` — componente de slot de imagem (drag-and-drop)
- `.image-slots.state.json` — fotos preenchidas nos slots
- `assets/` — logos

## Desenvolvimento

Por ser estático, basta abrir `index.html` no navegador ou servir a pasta:

```bash
npx serve .
```
